## 👉 С чего начать? 👈

Следуй по этим пунктам, и через 5 минут твой бот будет готов к работе:

1.  **Клонируй репозиторий** — Нажми зеленую кнопку `Code` и скачай архив или используй `git clone`.
2.  **Прочитай инструкцию** — 👈 **ЭТО САМЫЙ ВАЖНЫЙ ШАГ!** 👉 [ЧИТАТЬ TUTORIAL.MD ЗДЕСЬ](./tutorial.md) 👈
3.  **Настройка имени** — Зайди в `.github/workflows/main.yml`, убери решетку `#` в первой строке и дай боту имя.
4.  **Убери "Запинку"** — В том же файле в самом конце удали `exit 0`, чтобы включить автоматику.
5.  **Запуск** — Сделай `git push` и смотри, как GitHub Actions делает за тебя всю грязную работу.
